Das erste Mal mit Kitodo arbeiten

Kitodo wird Ihnen bei der ersten Nutzung sowohl optisch als auch in der Benutzung anders vorkommen. Daher folgt hier eine kurze Zusammenfassung, welche die anfängliche Nutzung von Kitodo unterstützen soll.

Ein Projekt ohne Produktionsvorlage anlegen

  1. Als erstes müssen Sie ein Projekt (falls noch nicht vorhanden) anlegen. Dazu klicken Sie im "Dashboard" auf "Projekte".
  2. Anschließend gelangen Sie auf die Seite Projekte. Dort klicken Sie auf den Button "Neu" und in der Schaltfläche auf "+ Neues Projekt".
  3. Auf der Seite Neues Projekt geben Sie die Daten für Ihr Projekt unter den 3 Reitern Details, Technische Daten, Mets Parameter ein.
  4. Klicken Sie anschließend auf "Speichern".
  5. Das neu angelegte Projekt erscheint nun auf der Seite Projekte in der Projektliste.

Einen Workflow für das Projekt erstellen

  1. Um einen neuen Workflow zu erstellen, klicken Sie auf den Button "Neu +" und anschließend auf " + Neuer Workflow"
  2. Sie gelangen auf die Seite "Neuen Workflow anlegen". Geben Sie dort zuerst den Titel für den Workflow ein.
  3. Nun können Sie mit den Symbolen auf der linken Seiten Ihren Workflo erstellen. Ziehen Sie dies Symbole einfach in die entsprechende Reihenfolge und Position.
  4. Achten Sie dabei darauf, dass immer Verbindungen zwischen den Symbolen vorhanden sind. Klicken Sie dazu auf das jeweilige Symbol und wählen die Verbindung aus und klicken auf das folgende Symbol.
  5. Durch einen Doppelklick auf das entsprechende Symbol können Sie diesen Namen geben und anschließend weitere Eigenschaften und Berechtigungen festlegen
  6. Wenn Sie alle Eingaben vorgenommen haben, wählen Sie den Status "Aktiv" aus und klicken Sie auf "Speichern".
  7. Anschließend wird Ihr erstellter Workflow unter dem Reiter "Workflows" angezeigt.

Eine Produktionsvorlage erstellen

  1. Um eine neue Produktionsvorlage zu erstellen, klicken Sie auf den Button "Neu +" und anschließend auf " + Neue Produktionsvorlage".
  2. Anschließend gelangen Sie auf die Seite "Neue Produktionsvorlage". Geben Sie dort zuerst den Titel für die neue Produktionsvorlage ein.
  3. Wählen Sie nun ein Projekt, einen Workflow, einen Regelsatz und einen Laufzettel für die Produktionsvorlage aus.
  4. Klicken Sie anschließend auf "Speichern".
  5. Ihre neue Produktionsvorlage wird nun unter dem Reiter Produktionsvorlagen angezeigt.

Eine Produktionsvorlage für das Projekt auswählen

Nachdem nun Projekt, Workflow und Produktionsvorlage erstellt sind, muss (falls noch nicht geschehen) für das neue Projekt noch die Produktionsvorlage ausgewählt werden:

  1. Klicken Sie im Bereich Projekte auf die Projektliste.
  2. Klicken Sie neben dem jeweiligen Projekt auf die Aktion "Bearbeiten.
  3. Klicken Sie auf den Reiter Produktionsvorlagen und entsperren Sie das Projekt, indem Sie auf den Button "Gesperrt" klicken.
  4. Klicken Sie nun auf den Button "Produktionsvorlage hinzufügen".
  5. Wählen Sie nun eine Produktionsvorlage aus, indem Sie auf das "+" neben der gewünschten Produktionsvorlage klicken.
  6. Die ausgewählte Produktionsvorlage erscheint nun in Ihrem Projekt unter dem Abschnitt Aufgabe.
  7. Schießen Sie nun noch das Pop-up-Fenster indem Sie auf "Schließen" klicken.
  8. "Speichern" Sie anschließend Ihre Änderung an dem Projekt.

Einen Vorgang anlegen

  1. Um einen neuen Vorgang anzulegen, klicken Sie in der Projektliste auf das Icon zum Ausklappen des jeweiligen Projekts.
  2. Nun klicken Sie auf das Icon "einen Vorgang auf Basis dieser Produktionsvorlage anlegen".
  3. Geben Sie unter den Reitern OPAC abfragen, Vorgangsdaten, Zusätzliche Details und Suche die Daten für den Vorgang ein.
  4. Anschließend finden Sie den neuen Vorgang in der Vorgangsliste oder auf Ihrem Kitodo-Desktop.
